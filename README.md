# Trader-Sentiment-Analysis
A data-driven analysis exploring how market sentiment (Fear vs Greed) influences trader performance, behavior, and risk-taking, with actionable insights and strategy recommendations.

## Summary of Insights
### Insight 1: Win Rate vs Sentiment
- Traders achieve the highest win rate during Extreme Greed (~46.7%), showing strong bullish trends improve trading success.
### Insight 2: Trading Activity vs Sentiment
- Trading activity peaks during Extreme Fear, showing more trades happen in volatile conditions.
### Insight 3: Profitability vs Sentiment
- Average profit per trade is highest during Extreme Fear, despite lower win rates.
### Insight 4: Trader Segmentation
- Frequent traders take larger positions (~5800 USD) and are more aggressive.
- Infrequent traders (~4393 USD) are more conservative.

## Tech Stack
- Python – data processing and analysis
- Pandas / NumPy – data manipulation
- Matplotlib / Seaborn – visualization
- Jupyter Notebook – project workflow
